# AI vs ML vs DL

## Introduction

Artificial Intelligence (AI), Machine Learning (ML), and Deep Learning (DL) are closely related fields. Deep Learning is a subset of Machine Learning, and Machine Learning is a subset of Artificial Intelligence.

---

## Relationship Between AI, ML, and DL

```text
AI
└── ML
    └── DL
```

## Artificial Intelligence (AI)

### Definition

Artificial Intelligence is a field of computer science that aims to create systems capable of performing tasks that normally require human intelligence.

### Examples

* Chatbots
* Robotics
* Expert Systems
* Virtual Assistants

### Key Points

* Broadest field
* Focuses on intelligent behavior
* Parent field of ML and DL

---

## Machine Learning (ML)

### Definition

Machine Learning is a subset of AI that enables systems to learn patterns from data without being explicitly programmed.

### Examples

* Spam Detection
* Recommendation Systems
* Price Prediction

### Key Points

* Learns from data
* Improves with experience
* Subset of AI

---

## Deep Learning (DL)

### Definition

Deep Learning is a subset of Machine Learning that uses deep neural networks to learn complex patterns from large amounts of data.

### Examples

* ChatGPT
* Face Recognition
* Self-Driving Cars
* Image Classification

### Key Points

* Uses Neural Networks
* Requires large datasets
* Subset of ML

---

## Comparison

| Field | Main Focus                          |
| ----- | ----------------------------------- |
| AI    | Smart behavior                      |
| ML    | Learning from data                  |
| DL    | Learning using deep neural networks |

---

## My Understanding

* AI is the broadest field.
* ML is a subset of AI.
* DL is a subset of ML.
* Every DL system is an ML system.
* Every ML system is an AI system.
* The reverse is not always true.
