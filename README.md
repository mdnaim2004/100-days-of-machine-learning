# 100 Days of Machine Learning

This repository contains notes, Jupyter notebooks, and small projects created while learning machine learning.

Repository layout

- Basic ML/: Collection of Jupyter notebooks and learning notes
- placement problem/: A small Flask app (`app.py`), dataset (`Placement.csv`), and templates/

Quick start

- To explore notebooks: run `jupyter lab` or `jupyter notebook` in the repository root and open files under `Basic ML/`.
- To run the placement demo app:
 1. Change into the app directory: `cd "placement problem"`
 2. Install dependencies: `pip install -r requirements.txt` (create one if missing)
 3. Run: `python3 app.py`
